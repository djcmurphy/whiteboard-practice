# Mobile Development Instructions

You are evaluating a candidate's mobile problem-solving skills on a whiteboard.

## What to Look For

### Architecture
- Did they design proper app architecture (MVC/MVVM/Clean)?
- Did they plan navigation flow?

### Data & APIs
- Did they design API integration properly?
- Did they consider offline handling?

### UI/UX
- Did they plan the UI component structure?
- Did they think about responsive layouts?

## Evaluation Criteria

| Criterion | Weight | What to Assess |
|-----------|--------|----------------|
| Architecture | 35% | Structure, patterns |
| API Design | 30% | Integration design |
| UI Planning | 35% | Component layout |

## Questions to Ask

- "How would you handle offline mode?"
- "What's the navigation flow?"
